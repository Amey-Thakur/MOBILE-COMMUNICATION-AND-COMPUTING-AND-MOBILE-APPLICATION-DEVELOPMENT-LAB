LIBRARY FILES
================================================================================

This directory contains the Java library dependencies required for running
the Path Loss experiment applet. These are third-party open-source libraries
used by the IIT Kharagpur Virtual Lab.

FILE                        VERSION     DESCRIPTION
--------------------------------------------------------------------------------
iText-2.1.5.jar             2.1.5       PDF generation and manipulation library
                                        Used for generating experiment reports
                                        License: MPL/LGPL
                                        Website: https://itextpdf.com/

jcommon-1.0.16.jar          1.0.16      Common utilities for JFreeChart
                                        Provides shared classes and utilities
                                        License: LGPL
                                        Website: http://www.jfree.org/jcommon/

jcommon-1.0.12.jar          1.0.12      Older version of JCommon (backup)
                                        Maintained for compatibility
                                        License: LGPL

jfreechart-1.0.13.jar       1.0.13      Java chart library for data visualization
                                        Used for plotting path loss graphs
                                        License: LGPL
                                        Website: http://www.jfree.org/jfreechart/

beansbinding-1.2.1.jar      1.2.1       JavaBeans Binding (JSR-295)
                                        Provides property binding between objects
                                        License: LGPL
                                        Website: https://beansbinding.dev.java.net/

                                        Prevents direct browsing of library folder

================================================================================
USAGE
================================================================================

These libraries are automatically loaded by the Java applet when running
the Path Loss experiment. No manual configuration is required.

The libraries are referenced in:
- Path_Loss.html (archive attribute)
- Path_Loss.jnlp (jar href elements)
